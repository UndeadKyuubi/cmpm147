Experiment 01 - Getting Started

Description

The program aims to generate fun stories of classic western duels, but throwing in some weird and wacky twists.

Technical

In this experiment, I altered the template message to fit the type of stories I wanted to generate and added my own fillers to use in the generation.

Reflection

Getting my own stories generated was a relatively simple process and I didn't really run into any major issues. What took the longest was just thinking about how I wanted to format the story and all of the different fillers I wanted to add.
